# 使用说明

## 首次部署

```bash
# 先编辑 pc-hotspot/scripts/hostapd.conf
bash pc-hotspot/scripts/setup-hotspot.sh
bash network-share/scripts/setup-network-share.sh
```

## 日常

```bash
~/bin/pc-hotspot-toggle.sh    # 开/关热点（NFS 随热点启停）
~/bin/open-share.sh           # 打开 ~/Share
```

## 手机：AzureNFS App

```bash
cd nfs-saf && ./build-and-install.sh
# App 内添加 <PC网关IP>:/export/azure-share
```

手机 → PC：复制到 `to-pc/`。

## 目录

| 目录 | 方向 |
|------|------|
| `from-pc/` | PC → 手机 |
| `to-pc/` | 手机 → PC |
